# 留白 Liubai

留白是一个面向自我觉察的记录产品。

它给你一个安静的空间，写下正在发生的事，也在合适的时候帮你回头看看：最近什么反复出现，哪些感受正在被看见，哪些珍视的东西正在慢慢变清楚。

留白不追求让你每天打卡，也不要求你一次说完整。你可以只写一点，只停一会儿，只把某个片段放下来。

## 你可以用它做什么

### 随手记

当你脑子里有一些还没整理好的想法，可以先写下来。

它适合：

- 记录当天的状态
- 放下一个反复出现的念头
- 保存一段不想马上解释的感受
- 在很累的时候，只留下几个关键词

### 看见

当某件事让你卡住、委屈、困惑或反复想起，可以用“看见”慢慢拆开。

它会从很简单的问题开始：

- 发生了什么？
- 当时最强烈的感受是什么？
- 身体有什么反应？
- 你真正迟疑的是什么？
- 这件事为什么对你重要？

你不需要回答得完整。写到哪里，都已经足够。

### 回顾

当你想看看最近这一段，可以进入回顾页。

它会把最近留下的内容轻轻放在一起：

- 反复出现的主题
- 正在珍视的东西
- 情绪气候
- 外面的天气
- 最近留下的痕迹

回顾不是给你评分，也不是替你总结人生。它只是帮助你看见：这段时间，你其实正在经历什么。

## AI 在这里做什么

留白里的 AI 更像一个专业教练，而不是诊断者。

它会：

- 帮你整理已经写下的内容
- 用温和的语言复述线索
- 提醒一些可能值得继续观察的地方
- 在回顾页生成阶段性总结

它不会：

- 给你贴心理标签
- 判断你做得好不好
- 替你下结论
- 要求你立刻改变

如果模型暂时不可用，页面也不会崩溃，系统会尽量用本地规则继续提供基础整理。

## 隐私与安全

留白处理的是非常私人的内容，所以产品设计遵守几个原则：

- 密码不会明文保存
- 模型 API Key 只应放在后端
- 不建议记录用户原文作为埋点
- 错误日志需要脱敏
- 回顾页刷新不会反复调用 AI，避免内容在用户没有主动选择时变化
